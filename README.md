Все необходимые зависимости перечислены в файле `requirements.txt`. Для их установки воспользуемся командой:
```bash
pip install -r requirements.txt
```

Перед установкой зависимостей можно создать виртуальное Python-окружение в директории с проектом командой:
```bash
python3 -m venv ./.venv
```

Запуск проекта осуществляется из директории проекта командой:
```bash
python3 ./main.py
```

При наличии файла `model.bin` с предобученной моделью в директории с проектом, запустится демонстрация работы модели. В случае отсутствия файла начнется обучение модели с последующим его созданием.
